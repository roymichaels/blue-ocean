## Build and self-host in minutes

The portal runs entirely as static assets generated by [VitePress](https://vitepress.dev/). Build it locally and serve it from
any CDN, bucket, or container platform:

```sh
yarn install
yarn docs:build
npx serve docs/.vitepress/dist --single --listen 4173
```

The build emits hashed assets that can be cached aggressively. For production, pin the build output to your preferred artifact
store and distribute the `/docs/.vitepress/dist` directory with an edge CDN or an S3-compatible bucket.

## Sample: publish a product to the marketplace topic

The following snippet creates a product message, signs it, and publishes it with the marketplace agent SDK.

```ts
import { createLightNode, waitForRemotePeer, Protocols } from '@waku/sdk';
import { MarketplaceAgent } from '@blue-ocean/sdk-near';

const TOPIC = '/blue-ocean/products/1';

async function publishProduct() {
  const node = await createLightNode({ defaultBootstrap: true });
  await node.start();
  await waitForRemotePeer(node, [Protocols.Relay]);

  const agent = new MarketplaceAgent({ node });
  await agent.authenticate({
    adminKey: process.env.ADMIN_PRIVATE_KEY!,
    workspace: 'alpha'
  });

  await agent.publishProduct(TOPIC, {
    id: 'hex-blade-9',
    name: 'Hex Blade 9',
    price: '49.00',
    currency: 'USD',
    availability: 'in-stock'
  });
}

publishProduct().catch(console.error);
```

> Use `.env.local` to store secrets like `ADMIN_PRIVATE_KEY`. The agent automatically canonicalizes and signs payloads before
> publishing them to the Waku network.
